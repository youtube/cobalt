# Cobalt First Time Setup

This workflow guides the agent through running the first-time setup script for Cobalt.

## Steps

1. **Run Script**: Run the script `//third_party/cobalt/cobalt_first_time_setup.sh` in a google3 workspace.
2. **Handle Interaction**: The script will ask for inputs like y/n, Name, and LDAP. Provide these inputs interactively.
3. **Self-Correction**: If the script fails, read the script file, analyze the error, modify the script to fix the issue, and retry.
4. **Next Steps**: Upon successful completion, suggest running the New Checkout workflow.
