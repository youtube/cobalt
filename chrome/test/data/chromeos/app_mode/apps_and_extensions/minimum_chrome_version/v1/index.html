App to Test minimum_chrome_version

This app has three versions:

Version 1.0.0 is a regular app and should always work.

Version 2.0.0 sets the `minimum_chrome_version` field in the manifest to an old Chrome version, so it should be installable in current Chrome versions.

Version 3.0.0 sets the `minimum_chrome_version` field in the manifest to a Chrome version M999, so it shouldn't be installable in Chrome versions for the foreseeable future.

This is version 1.0.0
