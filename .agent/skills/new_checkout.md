# Cobalt New Checkout Setup

This workflow guides the agent through running the new checkout script for Cobalt.

## Steps

1. **Run Script**: Run the script `cobalt_new_checkout.sh` at the root of the Cobalt repository.
2. **Handle Interaction**: The script will ask for inputs like y/n and GitHub username. Provide these inputs interactively.
3. **Self-Correction**: If the script fails, read the script file, analyze the error, modify the script to fix the issue, and retry.
